# Keybindings

Main modifier is `Option`/`Alt`. Omarchy bindings that use `Super+Alt` map to `Option+Command` on macOS.

## Launchers

```text
Super+Return              Open Ghostty
Super+Alt+Return          Open Ghostty attached to tmux main
Super+Space               Open Raycast
Super+Alt+Space           Show hyprmac help
Super+K                   Open this keybinding reference
Super+Escape              Toggle SketchyBar power menu
Super+B                   Open Safari
Super+Shift+B             Open Helium
Super+Shift+Return        Open Safari
Super+Shift+F             Open Finder
Super+Shift+N             Open Neovim in Ghostty
Super+Shift+D             Open LazyDocker in Ghostty
Super+Shift+O             Open Obsidian
Super+Shift+M             Open Spotify, fallback Music
Super+Shift+/             Open 1Password
Super+Shift+C             Open Calendar
Super+Shift+E             Open Mail
Super+Shift+A             Open ChatGPT
Super+Shift+G             Open Signal
Super+Alt+Shift+G         Open WhatsApp
Super+Shift+X             Open X
```

## Windows

```text
Super+W                   Close focused tile and terminate only that tile's process if no windows remain
Super+T                   Toggle floating/tiling
Super+O                   Pop out as floating/sticky/top layer
Super+L                   Toggle workspace layout bsp/stack
Super+J                   Toggle split direction
Super+F                   Toggle fullscreen zoom
Super+Ctrl+F              Toggle tiled/parent zoom
Super+Alt+F               Full-width approximation
Super+Arrow               Focus window directionally
Super+Shift+Arrow         Swap window directionally
Super+Alt+Arrow           Stack/group window directionally
Super+G                   Stack/group with recent window
Super+Alt+G               Move window out of stack/group
Super+Alt+Tab             Next stacked/grouped window
Super+Alt+Shift+Tab       Previous stacked/grouped window
Super+Alt+1..5            Focus stacked/grouped window 1..5
Super+S                   Show/toggle scratchpad
Super+Alt+S               Move focused window to scratchpad
Super+- / Super+=         Resize horizontally
Super+Shift+- / +=        Resize vertically
```

## Clipboard

```text
Super+C                   Copy via Cmd+C
Super+X                   Cut via Cmd+X
Super+V                   Paste via Cmd+V
Super+Ctrl+V              Raycast clipboard history
Super+Ctrl+E              macOS emoji picker
```

## Workspaces

```text
Super+1..9,0              Focus workspace 1..10
Super+Shift+1..9,0        Move window to workspace and follow
Super+Alt+Shift+1..9,0    Move window to workspace silently
Super+Tab                 Next workspace
Super+Shift+Tab           Previous workspace
Super+Ctrl+Tab            Recent workspace
Super+Alt+Shift+Arrow     Move current workspace to adjacent display
```

## System

```text
Super+Ctrl+L              Lock screen
Super+Ctrl+A              Sound settings
Super+Ctrl+B              Bluetooth settings
Super+Ctrl+W              Wi-Fi settings
Super+Ctrl+T              Activity Monitor
Super+Ctrl+C              macOS screenshot UI
Super+Ctrl+Space          Raycast background/menu placeholder
Super+Ctrl+Shift+Space    Raycast theme/menu placeholder
Super+Shift+Space         Toggle SketchyBar
Super+Shift+Backspace     Toggle gaps
Super+Ctrl+Backspace      Toggle padding
```

## Local Maintenance

```text
Super+Ctrl+Alt+R          Reload yabai, skhd, SketchyBar, borders
Super+Ctrl+Alt+Y          Restart yabai
Super+Ctrl+Alt+S          Restart skhd
Super+Ctrl+Alt+B          Restart SketchyBar/borders
Super+Ctrl+Alt+Arrow      Focus display prev/next
Super+Ctrl+Alt+Shift+Arrow Move window to display prev/next
```
